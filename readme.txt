Si se inicia el servidor con el comando:

    -node server 

    por defecto se iniciara en el puerto 8080 y modo fork.

Con -p y -m uno puede establecer el puerto y modo:

    -node server -p 8090 (se inicia el server en el puerto 8090)

    -node server -m CLUSTER (se inicia el server en modo cluster)

Comandos desafio:

    npm start (para 0x server.js)
    
    npm test (para node benchmark.js)
